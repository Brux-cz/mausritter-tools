## ğŸ¨ Design Philosophy

### **Visual Identity**
- **Color Palette:**
  - Primary: Earthy browns (#8B4513, #D2691E)
  - Secondary: Forest greens (#228B22, #6B8E23)
  - Accent: Gold (#FFD700) for highlights
  - Neutral: Beige (#F5F5DC), Cream (#FFFACD)
  - Dark: Charcoal (#36454F)
- **Typography:**
  - Headings: Geist Sans (bold, medieval-inspired)
  - Body: Geist Sans (regular)
  - Code: Geist Mono
- **Aesthetic:**
  - Hand-drawn illustrations (Stable Diffusion/Midjourney)
  - Parchment textures (subtle)
  - Minimal, functional layouts
  - Clear information hierarchy

### **Components**
- Using **shadcn/ui** components (Tailwind CSS)
- Custom theming for Mausritter aesthetic
- Accessible (WCAG AA)
- Responsive (mobile-first)

### Core Components Used:

**Layout:**
- `Card` - Containers for content sections
- `Tabs` - Navigation between views
- `Accordion` - Collapsible sections
- `Separator` - Visual dividers

**Forms:**
- `Button` - Primary actions
- `Input` - Text fields
- `Select` - Dropdowns
- `Radio Group` - Mutually exclusive options
- `Checkbox` - Toggle options
- `Textarea` - Multi-line text

**Data Display:**
- `Table` - Structured data (NPC lists, etc.)
- `Badge` - Status indicators (discovered, active, etc.)
- `Avatar` - User/character images
- `Progress` - HP bars, XP progress

**Feedback:**
- `Toast` - Notifications (success, error)
- `Dialog` - Modals (hex detail, take damage)
- `Alert` - Important messages
- `Tooltip` - Contextual help

**Navigation:**
- `Navigation Menu` - Top navigation
- `Breadcrumb` - Current location
- `Pagination` - List navigation
