# Brainstorm - Nápady a Zadání

Tento adresář slouží pro:
- Brainstorming nových funkcí a nástrojů
- Zadání pro generátory
- Nápady na vylepšení
- Diskuze o designu

## Struktura

- `ideas.md` - Obecné nápady a koncepty
- `generators.md` - Nápady na generátory
- `tools.md` - Nápady na nástroje
- `features.md` - Konkrétní funkce k implementaci
- `questions.md` - Otázky k řešení

## Jak používat

1. Zapisuj sem všechny nápady, i ty nedokončené
2. Odkazy na související dokumentaci
3. Náčrty API a struktur
4. Příklady použití
