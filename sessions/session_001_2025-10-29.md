## Implementované funkce

### Dice Roller (src/core/dice.py)
- `roll_d6()`, `roll_d20()`, `roll_d66()`
- `roll(dice: str)` - univerzální roller podporující "XdY" formát
- `roll_with_details()` - vrací i jednotlivé hody
- `attribute_test()` - roll-under mechanika
- `advantage_roll()`, `disadvantage_roll()`

### Data Models (src/core/models.py)
- `Character` - kompletní model postavy
- `Item` - předměty s slots, usage die
- `Condition` - stavy/kondice
- `Background` - pozadí postav
- `Location` - lokace
- `NPC` - non-player characters

### CLI (src/cli.py)

## Důležitá rozhodnutí

### Technologie
- **Jazyk**: Python 3.9+
- **CLI framework**: Click + Rich (pro barevný výstup)
- **Data models**: Python dataclasses (možná později Pydantic)
- **Testing**: pytest

### Architektura
- **Modulární struktura**: Oddělení core/generators/tools
- **Dokumentace**: Oddělení původních souborů od zpracovaných pravidel
- **Session tracking**: Markdown soubory pro udržení kontextu
- **Brainstorming**: Samostatný adresář pro plánování

### Datové formáty
- Export: JSON, Markdown (později možná PDF)
- Databáze: Zatím Python struktury, později JSON/YAML
- Konfigurace: pyproject.toml
